# 🍳 Eat-Map

Next.js와 Supabase를 활용하여 서울시 인증 식당 지도를 제작하였습니다. <br/>

## 🌟 주요 기능

### 🔐 SNS 로그인 기능

- next-auth를 이용한 카카오, 네이버, 구글 간편 가입 및 로그인

---

### 📌 카카오 지도를 이용한 맛집 위치 확인

- 메인 페이지에서 현재 위치로 이동 가능
- 지도 내 마커로 맛집 위치 확인 가능
- 특정 맛집 상세 정보 확인 기능

---

### ✍🏻 맛집 등록 / 수정 / 삭제
- 맛집 등록, 수정, 삭제 기능
- 맛집 리뷰 작성 및 작성한 리뷰 리스트 확인 가능

---

### 🔍 맛집 검색 및 찜 가능
- 맛집 목록 필터링, 정렬 기능
- 맛집 찜하기 기능 및 찜한 가게 리스트 확인 가능
