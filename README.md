# 🍳 Eat-Map

Next.js와 Supabase를 활용하여 서울시 인증 식당 지도를 제작하였습니다. <br/>

## 🌟 주요 기능

### 🔐 SNS 로그인 기능
<img width="856" height="472" alt="image" src="https://github.com/user-attachments/assets/1e8993f6-1fa7-453f-b8c9-74ea3f9de2fb" />

- next-auth를 이용한 카카오, 네이버, 구글 간편 가입 및 로그인

---

### 📌 카카오 지도를 이용한 맛집 위치 확인
<img width="856" height="472" alt="image" src="https://github.com/user-attachments/assets/0997ee56-dc87-4ed6-ba88-32d309df45bc" />

- 메인 페이지에서 현재 위치로 이동 가능
- 지도 내 마커로 맛집 위치 확인 가능
- 특정 맛집 상세 정보 확인 기능

---
