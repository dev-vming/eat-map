# Eat-Map

Next.js와 Supabase를 활용하여 서울시 인증 식당 지도를 제작하였습니다. <br/>
현재 kakao map API를 활용한 메인페이지, 맛집 목록 페이지를 개발 완료하였습니다.

## 🌟 주요 기능

+ [x] 카카오 지도를 이용한 맛집 위치 확인 (마커로 표시)
+ [x] 특정 맛집 상세 정보 확인 기능
+ [ ] 맛집 목록 필터링, 정렬 기능
+ [ ] 맛집 리뷰 작성 및 찜하기 기능
+ [ ] 찜한 가게 리스트 확인 기능
+ [ ] SNS 로그인 기능
